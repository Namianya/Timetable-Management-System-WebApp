HELP FACILITY

Computer science timetabling system helps in creating and managing of timetables. Interact with the system using links provided in left side. The following functions can be done:
Create Timetable
Click Create Timetable link, select units that will be on offer and lecturers who will be teaching then click done button that is located bottom of the selection form.
Edit Timetable
To edit timetable click on edit timetable link. Page with edit buttons will be loaded.In that page you can delete a unit using REMOVE UNIT button, add unit using ADD UNIT button and edit unit allocation using CHANGE ALLOCATION button.
View Timetable
To view timetable click on TimeTable link. A page will be loaded showing timetable.
View System Users
To view system users click on system users link. List of system users will be displayed there you can edit users details using the button provided.
Change Password
To change your password click Changepassword link. This loads a form where you will be required to fill correctly to change you password.
View Queries
To view lecturers' queries click Query link that loads recent queries submitted by lecturers.
